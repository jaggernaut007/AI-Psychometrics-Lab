# Attribution

## IPIP / IPIP-NEO-120 (Big Five)
This project includes item text from the International Personality Item Pool (IPIP) and uses the 120-item public-domain representation of the NEO Personality Inventory (“IPIP-NEO-120”). IPIP items and scales are intended to be public-domain personality measures.

Recommended references:
- International Personality Item Pool (IPIP): http://ipip.ori.org/
- Johnson, J. A. (2014). Measuring thirty facets of the Five Factor Model with a 120-item public domain inventory: Development of the IPIP-NEO-120. *Journal of Research in Personality*, 51, 78–89.
- IPIP citation guidance: https://ipip.ori.org/newCitation.htm

## MBTI
“MBTI” and “Myers‑Briggs Type Indicator” are trademarks of their respective owners.

This project does **not** include official MBTI® instrument items. Any type labels shown are computed using a **derived mapping from Big Five domain scores** (e.g., Extraversion → E/I, Openness → N/S, Agreeableness → F/T, Conscientiousness → J/P).

## DISC
This project includes a DISC-style word-choice inventory and scoring for research/educational use. It is not affiliated with or endorsed by any specific DISC publisher.
